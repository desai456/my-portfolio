Harshil Desai — Portfolio Website
A stunning, production-ready personal portfolio built with MERN Stack + Tailwind CSS.

🛠️ Tech Stack
Layer	Tech
Frontend	React 18 + Vite
Styling	Tailwind CSS v3
Charts	Recharts
Animations	CSS Animations + React Type Animation
Backend	Express.js + Node.js
Database	MongoDB + Mongoose
📁 Project Structure
Portfolio/
├── client/     # React + Vite frontend
└── server/     # Express + MongoDB backend
⚡ Quick Start
1. Install Client Dependencies
cd client
npm install
npm run dev
2. Install Server Dependencies
cd server
npm install
npm run dev
Open http://localhost:5173 to view the portfolio.

🌐 Sections
Hero — Animated particle canvas, typewriter effect, stats
About — Bio, education timeline, animated counters
Skills — Skill bars + radar chart + tech badges
Projects — 3D tilt cards with category filter
Experience — Hackathon & project timeline
Certifications — Glassmorphism cert cards
Contact — Form with Express/MongoDB backend
